1. Open a terminal
2. On the main directory for the project type npm i, this will install the needed modules for the project to run both backend and frontend
3. Next, type cd backend and type npm i, this will install the necessary modules for the backend
4. do the same thing for the frontend, type cd ../frontend, then type npm i to install frontend modules
5. go back to the main project directory
6. type npm start and it will show the url link of the project
7. ctrl + left click on the link to open the project.

***Note*** 
node v22.2.0 will have a DeprecationWarning but it will still run
recommend running the project on v20.11.00
